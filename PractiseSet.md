# 2 List any 4 difference between compiler and interpreter.:-4marks
- Compiler translates the entire source code into machine code (executable file) before execution, whereas interpreter translates and executes the code line-by-line.
- Compilation process detects all syntax errors at once, while interpretation stops at the first error encountered.
- Compiled programs generally run faster as they are already in machine code, but interpreted programs allow easier debugging and immediate execution.
- Compiler produces an intermediate object file or executable, whereas interpreter requires the source code to be present during runtime.

# 4 Explain what assembly language is and mention your its key characteristics.:-3 marks

**Assembly language** is a low-level programming language that uses mnemonic codes (like ADD, MOV) to represent machine instructions. It is a symbolic representation of machine code, one-to-one with processor instructions.

**Key characteristics:**
- It is low level language which is close to handware .
- It is different on different device/processor.It is written particularly dfor specific devices
- It is yet directly understandable by machine It is converted in to machine code by its program.
- It is very hard to debug for humans

# 9 Name any 4 implementation(flavours) of python . 4 Marks
- **CPython**: The standard and most widely used implementation, written in C.
- **PyPy**: JIT-compiled implementation for better performance using a Python interpreter.
- **Jython**: Runs on JVM, integrates Python with Java libraries and bytecode.
- **IronPython**: .NET implementation, runs on .NET runtime .
